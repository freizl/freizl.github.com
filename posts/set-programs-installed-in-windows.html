UP | HOME

Set programs installed in Windows Program files folder to PATH (Cygwin)

VIMHOME=’/cygdrive/c/Program Files/Vim/vim72’
JAVAHOME=’/cygdrive/c/Program Files/Java/jdk1.5.011’

alias ll=“ls -lh”
alias l=“ls -a”

PATH=“\({VIM\_HOME}":"\){JAVAHOME}/bin”:${PATH}
export PATH

The quote and double quote play the magic..
